# Professions

Profession-specific aliases for transmuter, reforger, golem master, and other advanced crafting roles.

> **In-game:** `library3kdb profession`

| Alias | Usage | Description | Location |
|---|---|---|---|
| `transmute_burn` | `transmute_burn` | Transmute crafting mats — run alone for more help | `modules/professions/transmuter.tin` |
| `transmute_ug` | `transmute_ug <jewel> {<quality>}` | Upgrade item to the next quality tier (curly braces around quality) | `modules/professions/transmuter.tin` |
| `transmute_ratios` | `transmute_ratios` | Show transmuter ratios for jewels | `modules/professions/transmuter.tin` |
| `refk` | `refk` | Buy a knife, reforge it, then dispose of it | `modules/professions/reforger.tin` |
| `refs` | `refs` | Buy a sword, reforge it, then dispose of it | `modules/professions/reforger.tin` |
| `ref` | `ref <item> <ACtype>` | Reforge an item to a specific AC defense type | `modules/professions/reforger.tin` |
| `refg` | `refg <item> <ACtype>` | Reforge an item on the ground | `modules/professions/reforger.tin` |
| `refk1` / `refk2` | — | Reforger knife sequence steps 1 and 2 | `modules/professions/reforger.tin` |
| `build_golem` | `build_golem <corpse name>` | Start a golem build sequence for a named corpse | `modules/professions/golem_master.tin` |
| `fill_golem` | `fill_golem` | Fill golem parts from held corpses | `modules/professions/golem_master.tin` |
