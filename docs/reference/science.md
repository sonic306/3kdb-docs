# Science Realm

Aliases for Science realm mechanics — Section Z level saves and Balangool crafting.

> **In-game:** `library3kdb science`

## Section Z — Level Saves

| Alias | Usage | Description | Location |
|---|---|---|---|
| `continue20` | `continue20` | Enter the level 20 continue code | `common/areas/sectionz.tin` |
| `continue40` | `continue40` | Enter the level 40 continue code | `common/areas/sectionz.tin` |
| `set20` | `set20` | Save yourself to level 20 | `common/areas/sectionz.tin` |
| `set40` | `set40` | Save yourself to level 40 | `common/areas/sectionz.tin` |

## Capsules

| Alias | Usage | Description | Location |
|---|---|---|---|
| `eat-caps1` | `eat-caps1` | Swallow one blue capsule (for Balangool crafts) | `common/areas/sectionz.tin` |
| `eat-caps2` | `eat-caps2` | Swallow blue and orange capsules | `common/areas/sectionz.tin` |
| `eat-caps3` | `eat-caps3` | Swallow blue, orange, and green capsules | `common/areas/sectionz.tin` |
