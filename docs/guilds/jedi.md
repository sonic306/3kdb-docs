# Jedi

The jedi guild package is a skeleton implementation. All standard guild files are present (including `hpbar.tin`), but no guild-specific aliases or strategies are defined beyond the placeholder.

The reference notes jedi have weapon mastery and lightsaber mode aliases (`weaponstats`, `setweaponmodetime`) in the library — these come from the library registration rather than the guild script itself.

See [Guild File Structure](./structure.md) to add jedi-specific automation.
