# Sii

Sii is a guild with minimal 3kdb-specific automation. The package provides GXP tracking reset and uses the common bot/action systems.

## Aliases

| Alias | Description |
|---|---|
| `gxp_reset` | Reset `total_rounds` and `total_gxp` counters |

## Script coverage

The Sii package contains a `gxp_reset` alias and an empty strategy/heartbeat skeleton. All combat is handled by the common action queue and bot system. Guild-specific abilities are not yet automated.
